Teleprompt app I made for DISCO @ St. Olaf College. Has a client and a display website. Client is able to upload or delete scripts. Display website does an auto-refresh to look for new scripts. It can then display it to a teleprompter it's hooked up to with options for text size, speed and centering. Pretty neat!

The original code is from **[Modern Hobbyist](https://www.youtube.com/channel/UCjgA1ehfjkZ4WMa5Cw9f1LA)**, altough the code has been heavily modified to fit our scope of the problem.
